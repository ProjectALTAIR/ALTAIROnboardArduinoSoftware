LTC4162 SDK
Version 1.0.0.0

The LTC4162 SDK allows users to quickly begin communicating with the LTC4162. The files in this package
are free to use in customer applications in compliance with the included license.

The SDK is broken into subdirectories by code language:

c_code:
    The c code files are broken into common files and example main files. To use the example files, copy
    the common files into the directory of one of the example files. Project-specific instructions are
    given in the README.txt file in the c_code directory.
